# CHE-616
This is a course on Introduction to Granular Mechanics.
For the course we have explored  through project the topic of  sound transmission in granular media through a multi-scale computational framework.
1) A hybrid modeling strategy combining the Discrete Element Method (DEM) and finite-difference wave propagation techniques is employed to capture both particle-scale interactions and continuum-level acoustic behavior.
2) The work focuses on understanding wave attenuation, dispersion, and energy transmission in saturated granular beds.
3) By bridging micro-mechanical contact dynamics with macroscopic wave response, the study provides insights relevant to geomechanics, soil acoustics, and advanced material design.
